Debug a Java app in Docker

Jun 5, 2017 · 1 minute read · Comments
dockerjava

To debug an application running inside a container run your container

$docker run -p 5005:5005 \
	-e JAVA_OPTIONS='-Xdebug -Xrunjdwp:transport=dt_socket,server=y,suspend=y,address=5005' \
	your_java_image

After that your application will stop and wait for a debugger. So, open your IDE and attach a debugger to Port: 5005.
